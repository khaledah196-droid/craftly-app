# 🏆 Craftly - Professional Social Network for Services

Craftly is a specialized social network connecting craftsmen, workers, service providers, and clients. Built with modern technologies for seamless communication, service discovery, and professional networking.

## 🌟 Features

- **Posts & Feed**: Share services, projects, and updates with images/videos
- **Direct Messages**: Real-time messaging between users
- **Follow System**: Build your professional network
- **Groups**: Create and join professional communities
- **User Profiles**: Showcase your work and credentials
- **Search**: Find craftsmen, services, and professionals
- **Advertisements**: Promote services with targeted ads
- **Premium Subscriptions**: Unlock advanced features
- **Post Boosting**: Increase visibility of your services
- **Multi-language Support**: Arabic (RTL), English (LTR), Hindi

## 🏗️ Tech Stack

### Web Frontend
- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling

### Mobile Frontend
- **Flutter** - Cross-platform mobile app
- **Dart** - Programming language

### Backend
- **Next.js API Routes** - RESTful API
- **Node.js** - Runtime

### Database & Storage
- **Supabase (PostgreSQL)** - Database
- **Supabase Storage** - File uploads
- **Supabase Realtime** - Real-time features

## 🎨 Color Scheme

- **Primary**: #4F46E5 (Indigo)
- **Secondary**: #22C55E (Green)

## 💰 Revenue Model

- **Feed Ads**: Display advertisements in user timelines
- **Post Boost**: Users can boost their posts for increased visibility
- **Premium Subscriptions**: Basic (100 EGP/month), Pro ($5/month)
- **Transaction Fee**: 7% commission on completed services (future)

## 👥 Target Audience

- Craftsmen (Plumbers, Electricians, Carpenters)
- Workers (Cleaners, Movers, etc.)
- Service Clients (Individuals seeking services)
- Companies (B2B service requests)
